# Meta Ad Specifications

## Image Sizes by Placement

| Placement | Dimensions | Aspect Ratio | Max File Size | Notes |
|-----------|-----------|--------------|---------------|-------|
| Feed (recommended) | 1080x1350 | 4:5 | 30 MB | Best performing; ~1% higher CTR |
| Feed (square) | 1080x1080 | 1:1 | 30 MB | Works across 80% of placements |
| Feed (hi-res portrait) | 1440x1800 | 4:5 | 30 MB | Retina displays |
| Feed (hi-res square) | 1440x1440 | 1:1 | 30 MB | Retina displays |
| Stories / Reels | 1080x1920 | 9:16 | 30 MB | Full-screen vertical |
| Right Column (desktop) | 1200x1200 | 1:1 | 30 MB | Desktop only |
| Audience Network | 1200x628 | 1.91:1 | 30 MB | External placements |
| Carousel (per card) | 1080x1080 | 1:1 | 30 MB | All cards must match |

**File formats:** JPG (90-100% quality recommended), PNG

## Stories/Reels Safe Zones

| Zone | Keep Clear |
|------|-----------|
| Top | 14% of canvas (profile info, timestamp) |
| Bottom | 20-35% of canvas (CTA, swipe-up, captions) |
| Sides | 6% each (edge bleed) |

## Character Limits

| Field | Visible | Maximum | Ideal |
|-------|---------|---------|-------|
| Primary Text | 125 chars | 2200 | <125 |
| Headline | 40 chars | 255 | <25 |
| Link Description | 30 chars | -- | <20 (carousel) |
| Carousel Primary | 125 chars | -- | 40-80 for mobile |

## Video Specs

| Spec | Requirement |
|------|-------------|
| Feed aspect ratio | 4:5 (recommended) or 1:1 |
| Stories/Reels ratio | 9:16 (1080x1920) |
| Max file size | 4 GB |
| Max duration (Feed) | 241 minutes |
| Optimal duration | 5-15s broad reach; 20-30s UGC demos |
| Codec | H.264 with AAC audio |
| Carousel video/card | Up to 15s, 4 GB max |

## Carousel Specs

| Spec | Requirement |
|------|-------------|
| Number of cards | 2-10 |
| Image size | 1080x1080 (1:1) or 1200x628 (1.91:1) |
| Image max file size | 30 MB per image |
| Video max per card | 15 seconds, 4 GB |
| Headline per card | 40 chars (25 ideal) |
| Description per card | 30 chars |

## Creative Fatigue Thresholds

| Signal | Threshold | Action |
|--------|-----------|--------|
| Frequency (cold) | >2.5 | Refresh creative or expand audience |
| Frequency (retargeting) | >4.0 | Rotate creative |
| CTR decline | >15% WoW | Creative fatigue likely |
| CPA increase | 2x historical | Meta flags "Creative Fatigue" |
| Refresh cadence (small audience) | Every 3-4 days | High-budget, narrow audience |
| Refresh cadence (broad audience) | Every 2-3 weeks | Larger audiences sustain longer |
| Meta recommendation | Every 7-10 days | Especially for specific/small audiences |
