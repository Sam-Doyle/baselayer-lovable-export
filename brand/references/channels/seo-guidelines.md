# SEO Guidelines

## Schema Types (JSON-LD Only)

| Content Type | Schema Type | Required Fields |
|-------------|------------|----------------|
| Product page | Product + Offer | name, price, availability, brand, image, description |
| Blog post | BlogPosting | headline, author, datePublished, image, publisher |
| Ingredient page | WebPage + MedicalEntity | name, description |
| Skin concern | WebPage + MedicalCondition | name, description |
| Homepage | Organization + WebSite | name, url, logo, sameAs |
| About page | AboutPage | name, description |
| Comparison page | WebPage + Product (x2) | name, description, product details |

## Schema Rules
- JSON-LD only (no microdata, no RDFa)
- No FAQ schema, no HowTo schema
- Product schema must include Merchant Center fields (price, availability, brand)
- Schema rich snippet CTR uplift: 20-30%+
- Product schema with reviews CTR uplift: 30%+

## Core Web Vitals Targets

| Metric | Good | Needs Improvement | Poor |
|--------|------|-------------------|------|
| LCP (Largest Contentful Paint) | <2.5s | 2.5-4.0s | >4.0s |
| INP (Interaction to Next Paint) | <200ms | 200-500ms | >500ms |
| CLS (Cumulative Layout Shift) | <0.1 | 0.1-0.25 | >0.25 |

CWV carry 25-30% ranking weight for competitive queries. Sites meeting all 3 thresholds see 8-15% visibility boost.

| Keyword Type | Volume | KD Range | Strategy |
|-------------|--------|----------|----------|

| Commercial ("buy men's moisturizer") | 500+ | 20-40 | Product/transactional pages |

Target: KD <40 with search volume >500/month for early wins.
